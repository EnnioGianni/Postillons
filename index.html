Bienvenue sur mon site de Marques Postales

Sur ce site vous trouverez la cotations de toutes les marques postale des origines à 1791

Cotes des marques postales

Celles-ci s'entendent pour de belles frappes.

Pour les lettres en port paye:
  1. Cote du PP sans plus value si la marque de port (dû) qui l'accompagne est d'un indice inférieur à 16.
  2. Si la marque de port (dû) qui accompagne le (port payé) est d'un indice supérieur a 16, majorer de 2 indices la cote du port payé
  3. Pour les marquez plus ancienne sans marque allez sur le lien LettresAnciennes
  4. Sur la page d'accueil cliquez sur une lettre pour vous rendre sur le sommaire de toute les villes qui commence par cette lettre
